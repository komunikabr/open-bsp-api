Recursos

Tudo que você precisa

Uma stack completa para construir produtos de mensageria com WhatsApp e Instagram de forma profissional.

Multi-tenant nativo

Cada organização tem seu próprio namespace isolado com RLS do Postgres. Dados 100% segregados sem código adicional.

API Key + JWT

Dois modos de autenticação que coexistem. Usuários humanos via JWT, AI agents e integrações via API Key.

AI-ready (MCP + A2A)

Model Context Protocol e Agent-to-Agent protocol nativos. Seus AI agents se conectam à plataforma sem código extra.

Edge Functions Deno

Webhooks, dispatcher, gerenciamento e pré-processamento de mídia rodando em Edge Functions com latência mínima.

Billing integrado

Schema de billing completo: planos, tiers, uso por produto, créditos IA, faturas e pagamentos. Pronto para monetizar.

Painel Admin

Interface administrativa para gerenciar organizações, planos, produtos, tiers, faturas e administradores da plataforma.

Como funciona

Em 3 passos

1

Crie sua organização e conecte o WhatsApp

Registre-se no portal, crie sua organização e vincule seu número WhatsApp Business via Meta Business API. Suporte a múltiplos números por org.

Integre via API Key em minutos

Gere uma API Key no portal e comece a enviar mensagens, gerenciar conversas e receber webhooks em tempo real. SDK disponível para JS e Python.

Conecte seus AI Agents via MCP

Use o protocolo MCP para dar aos seus LLMs acesso direto às conversas, mensagens e contatos. Compatível com Claude, GPT-4 e qualquer agent A2A.
